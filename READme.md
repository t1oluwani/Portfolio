# Tioluwani Akinloye's Portfolio

Welcome to my portfolio! This document provides an overview of the portfolio website, including its features, technologies.

## Overview

This portfolio is designed to showcase the skills, projects, and experiences of Tioluwani Akinloye, a multidisciplinary developer and designer. The website utilizes React.js HTML and CSS, with react libraries for responsive design and Font Awesome for icons.

## Features

### Sections

- **Home**: 
  - Introduces me with a brief description and a call-to-action button to get started.
  
- **About**: 
  - Offers a detailed biography, including educational background, personal interests, and contact links.
  - Features social media icons for easy access to my profiles and a link to download my resume.

- **Skills**:
  - Displays a grid of my technical skills with icons representing various technologies and tools. This includes programming languages, development environments, and more.

- **Projects**:
  - Showcases my recent projects with images, titles, and links to view them live or access their GitHub repositories.

- **Experiences**:
  - Highlights my professional and educational experiences.

- **Contact Me**:
  - Provides multiple ways to get in touch including email, a Calendly link for scheduling meetings, and a contact form for direct communication.


## Technologies Used

- **HTML**: Provides the structure of the portfolio.
- **CSS**: Used for styling, with an external stylesheet (`styles.css`) to customize the look and feel.
- **ReactJS**: Utilized for interactive components and managing state within the portfolio.
- **Font Awesome**: Icon library for social media and other icons.
